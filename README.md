TackleIndex 〜釣具モデル検索〜
================================

TackleIndex は、釣具の番手・モデル番号を効率的に検索できる無料のウェブツールです。
メーカーの公式サイトや通販ページでは探しづらい「過去モデル」「型番一覧」を横断的に確認することができ、アングラーが必要な情報にすばやくアクセスすることを目指しています。

主な特徴
--------
- ロッド・リールの番手一覧を体系的に掲載
- メーカー別 / カテゴリ別の階層ナビゲーション
- 過去モデルを含めた型番の整理
- スマートフォンでも見やすいUI
- 番手をクリックすると Google 検索に即リンク

使い方
------
1. 左サイドバーからカテゴリを選択  
2. メーカー → 系列 → モデルの順に絞り込み  
3. モデル名をクリックすればそのまま Web 検索できます
